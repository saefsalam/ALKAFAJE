## 🔧 مشاكل محتملة وحلولها

### مشكلة: "permission denied for table X"
**السبب:** RLS مفعّل ولكن لا توجد سياسة تسمح بالوصول
**الحل:** 
```sql
-- تحقق من السياسات الموجودة
SELECT * FROM pg_policies WHERE tablename = 'X';

-- أضف سياسة مناسبة
```

### مشكلة: "لا تظهر المنتجات"
**السبب:** المنتجات غير نشطة أو محذوفة
**الحل:**
```sql
-- تحقق من حالة المنتجات
SELECT id, title, is_active, is_deleted FROM items LIMIT 10;
```

### مشكلة: "لا تظهر الطلبات في الداشبورد"
**السبب:** الموظف غير مرتبط بالـ shop_id الصحيح
**الحل:**
```sql
-- تحقق من ارتباط الموظف
SELECT * FROM shop_users WHERE user_id = 'USER_ID_HERE';
```

### مشكلة: "العميل لا يستطيع إنشاء طلب"
**السبب:** customer_id غير صحيح
**الحل:**
```sql
-- تحقق من ارتباط العميل
SELECT * FROM customers WHERE auth_user_id = 'AUTH_USER_ID';
```

## 📊 التحقق النهائي

### استعلام للتحقق من تفعيل RLS:
```sql
SELECT tablename, rowsecurity 
FROM pg_tables 
WHERE schemaname = 'public' 
AND rowsecurity = true;
```

### استعلام للتحقق من السياسات:
```sql
SELECT tablename, policyname, cmd 
FROM pg_policies 
WHERE schemaname = 'public'
ORDER BY tablename;
```
